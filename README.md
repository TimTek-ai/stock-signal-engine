# Stock Signal Engine

A personal side project exploring how market news and data can be processed into simple trading signals.

## Goals
- Learn practical data engineering and ML concepts
- Build a realistic signal pipeline (not fake predictions)
- Gain experience relevant to quantitative finance

## Features (Planned MVP)
- News ingestion from public sources
- Basic sentiment / keyword analysis
- Simple rule-based signal output (bullish / bearish / neutral)

## Tech Stack
- Python
- Pandas / NumPy
- Git + GitHub
- Ubuntu / Windows dev setup

## Disclaimer
This project is for educational purposes only and does not provide financial advice.
